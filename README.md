Qu'est-ce que Léon Acide [EXP] ?
C'est un outil de suppression de fichiers inutiles et personnalisations (de la performance).


Pourquoi et est-ce bien utile ?

• Le gain potentiel des ressources matérielles est tout simplement évident, on note une amélioration de 30%.

• Une protection certaine contre les attaques dirigées par la firme logicielle au travers du système de mise à jour.

• Suppression d'images contrefaites, on note une amélioration de 95%.


Mot de passe (extraire avec 7zip) : LeonAcide



Comment utiliser ce script .batch ? Extraire Léon Acide [EXP] dans un dossier*, faire un "clic" avec votre souris sur le fichier "(Installer)", puis faire un "clic" sur le fichier (démarrer).


*pour les utilisateurs qui souhaite réinstaller le .batch après avoir utilisés le patch "Fixer les permissions Windows", créer un dossier "LeonAcide", en accordant le privilège, à la racine de votre disque "C:".
